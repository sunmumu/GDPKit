# GDPKit

ios项目开发常用方法,整理和封装,例如YYKit框架.
一直持续整理更新.


## Use
法一 cocopods引用
pod 'GDPKit'

法二 文件若如项目中
下载GDPKit框架,把GDPKit文件夹拷贝到项目中.
导入头文件#import "GDPKit.h".

## 调用方法
大部分是类方法, 方便直接调用

### 例
```
NSString *version = [NSString getAppVersion];
```
